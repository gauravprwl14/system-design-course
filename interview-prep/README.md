## üí° Interview Tips

### Answering System Design Questions
1. **Clarify requirements** (5 minutes)
2. **High-level architecture** (10 minutes)
3. **Deep dive components** (15 minutes)
4. **Discuss trade-offs** (5 minutes)
5. **Scale and optimize** (10 minutes)

### Answering Technical Questions
1. **Quick answer** (30 seconds)
2. **Expand with context** (1-2 minutes)
3. **Provide example** (1 minute)
4. **Discuss alternatives** (30 seconds)

### Common Mistakes to Avoid
- ‚ùå Jumping to implementation without clarifying
- ‚ùå Not discussing trade-offs
