# 12-interview-prep/system-design/ — Layer 2 Router

Structured system design interview questions organized by difficulty and domain — from API and caching fundamentals through real-time systems, storage, messaging, business patterns, and AI/LLM systems.

## Subsections

| Folder | Difficulty | Description |
|--------|------------|-------------|
| fundamentals/ | Intermediate | Core API, caching, rate limiting, and circuit breaker questions |
| storage-and-databases/ | Intermediate–Advanced | Database, file system, search, and geospatial questions |
| messaging-and-streaming/ | Intermediate–Advanced | Message queues, event-driven, social feed, and streaming questions |
| scale-and-reliability/ | Intermediate–Advanced | CDN, Kubernetes, microservices migration, and observability questions |
| real-time-systems/ | Advanced | WebSockets, live streaming, video conferencing, and collaborative editing |
| business-and-advanced/ | Advanced | E-commerce, fraud detection, recommendations, ads, and saga/CQRS patterns |
| ai-and-agents/ | Advanced | LLM APIs, RAG, agent loops, multi-agent coordination, and prompt injection |

## Routing Table

| Task / Question | Go to |
|-----------------|-------|
| Start interview prep from the beginning | fundamentals/ |
| Practice database design questions | storage-and-databases/ |
| Practice Kafka or event streaming questions | messaging-and-streaming/ |
| Practice scalability and deployment questions | scale-and-reliability/ |
| Practice WebSocket or streaming questions | real-time-systems/ |
| Practice payment or booking system questions | business-and-advanced/ |
| Practice AI system design questions | ai-and-agents/ |
